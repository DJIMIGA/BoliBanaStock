# Assets BoliBana Stock Mobile

Ce dossier contient tous les assets visuels de l'application mobile BoliBana Stock.

## Logos et Icônes

### Fichiers SVG (Sources)
- `logo.svg` - Logo principal haute résolution
- `icon.svg` - Icône principale de l'application
- `adaptive-icon.svg` - Icône adaptative pour Android
- `splash-icon.svg` - Icône pour écran de démarrage
- `favicon.svg` - Favicon pour navigateurs

### Fichiers PNG (À générer)
Ces fichiers doivent être générés à partir des SVG. Voir `GUIDE_GENERATION_LOGOS.md` pour les instructions.

- `icon.png` (1024x1024) - Icône principale
- `adaptive-icon.png` (1024x1024) - Icône adaptative
- `splash-icon.png` (512x512) - Icône splash screen
- `favicon.png` (256x256) - Favicon

## Génération des icônes

Consultez le fichier `GUIDE_GENERATION_LOGOS.md` pour les instructions détaillées sur la conversion des SVG en PNG.

## Design

Le logo utilise les couleurs de la marque BoliBana Stock :
- **Bleu principal** : #2B3A67
- **Or** : #FFD700
- **Vert forêt** : #2E8B57

Le design représente la gestion de stock avec des boîtes empilées et un graphique de croissance.
