=== Installation de BoliBana Stock ===

1. Copier le dossier "bolibanastock" sur le bureau ou dans le dossier de votre choix.
2. Double-cliquer sur "run.bat" (Windows) ou exécuter "run.sh" (Linux/macOS).
3. Ouvrir votre navigateur à l'adresse : http://127.0.0.1:8000
4. Pour arrêter l'application, fermez la fenêtre de commande.

=== Mise à jour ===
Pour mettre à jour, remplacez le dossier "bolibanastock" par la nouvelle version.

=== Export/Import des données ===
Les données sont stockées dans le fichier "app/db.sqlite3".

=== Structure des dossiers ===
- app/ : Code source de l'application Django
- static/ : Fichiers statiques (CSS, JS, images)
- venv/ : Environnement virtuel Python
- requirements.txt : Liste des dépendances
- run.bat : Script de lancement Windows
- run.sh : Script de lancement Linux/macOS
