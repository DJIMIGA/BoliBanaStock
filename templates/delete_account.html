BoliBana Stock - Demande de suppression de compte et données associées

Comment demander la suppression de votre compte

Pour demander la suppression de votre compte et de toutes les données associées, veuillez suivre les étapes suivantes :

  1. Depuis l'application mobile :
    • Ouvrez l'application BoliBana Stock
    • Allez dans Paramètres → Profil utilisateur
    • Cliquez sur Supprimer mon compte
    • Confirmez votre demande
  2. Par email :
    • Envoyez un email à support@bolibanastock.com
    • Indiquez votre nom d'utilisateur ou email de compte
    • Précisez que vous souhaitez supprimer votre compte
    • Nous vous répondrons dans les 48 heures
  3. Par téléphone :
    • Appelez le +223 72 46 42 94
    • Demandez la suppression de votre compte
    • Préparez votre nom d'utilisateur pour vérification
⚠️ Important : La suppression de votre compte est définitive et irréversible. Assurez-vous d'avoir exporté toutes les données importantes avant de procéder.

Données qui seront supprimées

Lors de la suppression de votre compte, les données suivantes seront définitivement supprimées :

  • Données de compte : Nom d'utilisateur, email, profil utilisateur
  • Données personnelles : Nom, prénom, numéro de téléphone
  • Préférences : Paramètres de l'application, préférences utilisateur
  • Données opérationnelles : Si vous êtes le seul utilisateur de votre entreprise, les données suivantes peuvent être supprimées :
    • Produits et catalogues
    • Historique des ventes
    • Clients et programmes de fidélité
    • Rapports et analyses

Données qui seront conservées

Certaines données peuvent être conservées pour des raisons légales ou opérationnelles :

  • Données financières : Les transactions financières peuvent être conservées selon les obligations légales (généralement 7 à 10 ans)
  • Journaux d'audit : Les journaux d'activité peuvent être conservés pour des raisons de sécurité (jusqu'à 12 mois)
  • Données anonymisées : Les données agrégées et anonymisées peuvent être conservées à des fins statistiques
ℹ️ Période de rétention : Après votre demande de suppression, vos données seront supprimées dans un délai maximum de 30 jours. Certaines données peuvent être conservées plus longtemps selon les obligations légales applicables.

Suppression partielle de données

Vous pouvez également demander la suppression de certaines données spécifiques sans supprimer votre compte complet :

  • Suppression de votre historique de ventes
  • Suppression de vos clients
  • Suppression de vos produits
  • Modification de vos données personnelles

Pour ces demandes, contactez-nous par email à support@bolibanastock.com en précisant les données que vous souhaitez supprimer.
